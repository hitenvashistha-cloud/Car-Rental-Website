# 🚗 KarZone - Car Rental Website

A full-stack car rental platform built with the MERN stack (MongoDB, Express.js, React.js, Node.js). Users can browse luxury cars, make bookings, and car owners can list their vehicles for rent.

## ✨ Features

### For Users
- Browse luxury cars with filters (brand, price, seats)
- View detailed car specifications
- Book cars with date selection and automatic price calculation
- View booking history with status tracking

### For Owners
- List new cars with complete details
- Manage own car listings (edit/delete)
- View incoming booking requests
- Confirm or cancel bookings
- Track total revenue

### General
- Secure authentication using JWT and bcrypt
- Role-based access control (User vs Owner)
- Responsive dark-themed UI with animations
- Double-booking prevention using date overlap validation

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js, React Router, Axios, Context API, CSS3 |
| Backend | Node.js, Express.js, JWT, bcryptjs |
| Database | MongoDB, Mongoose |
